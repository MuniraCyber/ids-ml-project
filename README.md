# Intrusion Detection System (IDS) 🛡️

A Machine Learning-based Intrusion Detection System that classifies network traffic 
as normal or malicious using the NSL-KDD dataset.

## Features
- 🤖 Random Forest classifier with 99.83% accuracy
- 🌐 Interactive web interface built with Streamlit
- 📊 Real-time network traffic classification
- 🔍 Detects multiple attack types (DoS, Probe, R2L,
